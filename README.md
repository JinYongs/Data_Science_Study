- 기본적으로 개인주제(데분관련 플젝), Kaggle, Dacon, 공공데이터 등 1주일 간 본인이 공부한 것을 발표하는 것으로 진행합니다.
- 매주 발표하는 내용을 서로 피드백하고, 의견을 자유롭게 공유하면서 공부해 나가는 방식입니다.

- 일정 분기가 끝나면 (개인 진행 사항이 어느정도 정리되면) Git or 포토폴리오로 문서화를 진행 합니다.
